# 📂 GitHub Actions & PR Template Demo

A simple GitHub repository demonstrating how to:

- Create a **Pull Request (PR) template**
- Set up **CI/CD pipelines** using **GitHub Actions**

This project shows how you can automate code quality checks and standardize PRs for better team collaboration.

---

## ✨ Features

✅ **GitHub Actions** — Runs tests automatically on every push and pull request.

✅ **Pull Request Template** — Provides a clear, reusable PR description checklist.

---

## 📌 What’s inside

- `.github/workflows/ci.yml` — Defines the CI pipeline.
- `.github/PULL_REQUEST_TEMPLATE.md` — PR template used automatically when you open a pull request.

---

## 🧩 How to use

1. **Clone the repo**
2. Check out `.github` folder for configuration examples
3. Customize for your own projects

---

## ⚙️ Tech

- **CI/CD** — Node.js example with `npm test`
- **GitHub Actions** — Native GitHub workflow
- **PR Templates** — Native GitHub feature
